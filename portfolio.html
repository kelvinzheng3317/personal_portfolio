What I've made so far...

Wordle

Recreated the popular word web game Wordle using pure HTML, CSS, and Javascirpt. Includes a useable on screen keyboard.

2048

Recreated the popular word web game 2048 using pure HTML, CSS, and Javascript. Fully playable and keeps track of high score. Utilizes local storage to maintain high-score even after refresing page.

Weather Data Analyzer

Takes in weather data from an outside file, performs calculations and outputs results visually through graphs. The program also allows the user to enter a month and year to see the statistics for that month. Uses Python as well as numpy and matplotlib modules.

Etch-a-sketch

Created a web version of Etch-a-sketch using HTML, CSS, and Javascirpt. Includes the ability to choose brush color, brush size, change grid units, and reset drawing.

Tic-tac-toe game

Tic-tac-toe game built using Python. User can play against a second player or against a bot. Uses object oriented programming.

Sudoku solver

A recursive algorithm that can solve any Sudoku puzzles. Build using C++.
